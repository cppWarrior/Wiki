### 1. 스타일 가이드

- 대단원 명 앞에는 #, 중단원 명 앞에는 ##, 소단원 명 앞에는 ###을 붙여줍니다.

- 코드블럭은 \`\`\`cpp, \`\`\`로 감쌉니다. \`는 숫자 1 왼쪽에 있는 기호입니다.

```cpp
#include <iostream>

using namespace std;

int main()
{
    cout << "hello world" << "\n";
}
```
(raw를 클릭하여 원본을 봐보세요.)

- 코드에서 들여쓰기는 4개의 공백 합니다.

- 작업이 끝나면 중단원별로 진행상황을 기록해주세요.(표 아직 안 만들어짐)
